# knowledge

`knowledge` 保存已经整理过、可长期复用的人类可读知识。

## Directories

```text
notes      阶段性判断、任务总结、可复用观察
projects   活跃项目状态和约束
processes  可复用操作流程
concepts   稳定原则、长期概念、判断框架
skills     文档态 skill，不等同于 runtime skill 安装目录
.claims    大文档的 claim sidecar JSON
```

`knowledge` 是长期知识的 source of truth。外部原始资料放在 `library`，审核队列放在 `dream/review`。
