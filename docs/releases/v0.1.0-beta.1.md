# A-Brain v0.1.0-beta.1

First trial release of A-Brain.

## What is included

- Seven first-class modules: `diary`, `library`, `ingest`, `knowledge`, `think`, `dream`, `learn`
- Windows PowerShell command pairs for the main workflows
- `AGENTS.example.md` for agent-loop integration
- Sanitized `examples/` for first-run testing
- `scripts/smoke-test.cmd` for release verification in a temporary copy

## Intended audience

- AI heavy users
- Agent-first local knowledge system builders
- People who want a human-readable memory layer around coding agents

## What this beta is for

- Validate the module split
- Validate the Windows-first command surface
- Validate that new users can clone, wire, and run the system

## What is intentionally not in scope yet

- Full semantic embedding backend
- Universal multi-agent session import coverage
- Automatic runtime skill installation
- Private adapters and private knowledge overlays

## Upgrade notes

- The text-index mode is labeled `local-text`
- Local private config files may live at `config/*.local.json`, but v1 commands still read `config/a-brain.json` directly
